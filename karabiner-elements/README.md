# Karabiner Elements configuration

Configuration files for the Karabiner Elements MacOS application to remap and
customize your keyboard.

## How to use

Pick a rule you want to use amongst the ones available in this directory.
For example, pick `caps_lock-ctrl-esc.json` to remap Caps Lock as both Escape
and, when you hold it, as Control.

In Karabiner Elements, go to `Complex Modifications`, create a new one, and
copy / paste the content of the rules you want to apply.
